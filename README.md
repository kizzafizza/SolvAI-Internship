# SolvAI Internship

AI-powered educational feedback system developed during my internship at SolvAI.

## Overview

This project explores the development of an intelligent AI feedback engine designed to analyse student responses and generate a personalised, actionable educational feedback for individual students.

The system aims to help students improve their answers by identifying:

- What is correct
- What is missing
- How responses can be improved

---

## Project Goals

- Build an AI-powered feedback generation pipeline
- Experiment with prompt engineering and agentic workflows
- Evaluate AI-generated feedback quality
- Iteratively improve model performance based on testing
- Develop a baseline AI educational assistant

---

## Features

- Student answer analysis
- Automated feedback generation
- Prompt-based / agentic AI evaluation
- Feedback quality assessment pipeline
- Modular architecture for experimentation
