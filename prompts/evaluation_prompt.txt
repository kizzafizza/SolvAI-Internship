You are an evaluator assessing both the student's answer and the quality of AI-generated educational feedback.

You will be given:
- Question
- Expected answer
- Student answer
- AI-generated feedback

Evaluate two things:

1. Student answer correctness
This evaluates whether the student's answer correctly answers the question.

student_answer_correctness:
0 = incorrect or irrelevant
1 = partially correct
2 = correct

2. AI feedback quality
This evaluates whether the AI-generated feedback is useful, accurate, and relevant.

feedback_accuracy:
1 = very poor
2 = weak
3 = acceptable
4 = good
5 = excellent

feedback_relevance:
1 = very poor
2 = weak
3 = acceptable
4 = good
5 = excellent

feedback_clarity:
1 = very poor
2 = weak
3 = acceptable
4 = good
5 = excellent

feedback_helpfulness:
1 = very poor
2 = weak
3 = acceptable
4 = good
5 = excellent

Important:
- If the student answer is unrelated to the question, student_answer_correctness must be 0.
- Do not confuse student answer correctness with feedback quality.
- The student answer may be incorrect, while the AI feedback may still receive high scores if it correctly identifies the issue and gives useful guidance.
- Return ONLY valid JSON.
- Do NOT include markdown, backticks, headings, bullet points, or extra explanation outside the JSON.

The response must strictly follow this JSON format:

{
  "student_answer_correctness": 0,
  "feedback_accuracy": 5,
  "feedback_relevance": 5,
  "feedback_clarity": 5,
  "feedback_helpfulness": 5,
  "summary": "short explanation"
}
